Project Overview

The Future Smart Farm project is a comprehensive solution that applies the latest technologies to build an automated, precise, and efficient farm management system. The system's key innovation lies in its ability to intelligently combine two powerful data streams: real-time environmental data collected directly from the farm and predictive data from the NASA GMAO model (via Open-Meteo). By synthesizing these two streams of information, the system automatically provides optimal solutions and farming choices, enabling proactive, efficient operations that are ready to adapt to weather changes.

Technology Stack

Cloud Data & APIs

Leverages powerful APIs like the NASA GMAO Model via Open-Meteo to fetch real-time and forecasted weather data. This includes crucial parameters like temperature, humidity, precipitation, and solar radiation.

Python

Used to write powerful scripts that automatically collect data from the cloud APIs and insert it into the local database for processing and display.

PHP & MySQL (XAMPP)

Provides a solid backend to pull data from the MySQL database and serve it to the frontend, while also handling user authentication and other server-side tasks.

HTML / CSS / JS

Builds the modern user interface that displays data fetched by PHP and sends user commands via JavaScript to the control layer of the system.

Node-RED

Acts as the central messaging and logic hub. It receives commands from the web interface and translates them into instructions for the PLC hardware.

PLC S7-1200

The on-site control brain that reliably executes commands from Node-RED, controlling physical hardware like pumps, fans, and lights with industrial precision.

Data & Reference Sources

NASA Earth Science Data

Provides free access to one of the world's largest archives of Earth science data, including satellite observations and climate models like GMAO, which are fundamental to this project's predictive capabilities.

NASA Space Apps Challenge

The project was inspired by and developed for the world's largest global hackathon, which encourages the use of NASA's open-source data to address real-world challenges on Earth and in space.

Vietnam Academy of Agricultural Sciences (VAAS)

A leading research institution providing in-depth knowledge of agricultural science, animal husbandry, and cattle breeds in Vietnam.

Ministry of Agriculture & Rural Development

Provides official information on policies, markets, and advanced farming and animal husbandry techniques.

Food and Agriculture Organization (FAO)

Offers global data, analysis, and reports on agriculture, food security, and the cattle industry.

Scientific Journals & Articles

Academic materials from reputable sources like the Journal of Integrative Agriculture, Agricultural Journal, and the ScienceDirect database.
