Project Overview

The Future Smart Farm project is a comprehensive solution that applies the latest technologies to build an automated, precise, and efficient farm management system. The system's key innovation lies in its ability to intelligently combine two powerful data streams: real-time environmental data collected directly from the farm and predictive data from the NASA GMAO model (via Open-Meteo). By synthesizing these two streams of information, the system automatically provides optimal solutions and farming choices, enabling proactive, efficient operations that are ready to adapt to weather changes.

Technology Stack

Cloud Data & APIs

Leverages powerful APIs like the NASA GMAO Model via Open-Meteo to fetch real-time and forecasted weather data. This includes crucial parameters like temperature, humidity, precipitation, and solar radiation, providing the critical input for predictive analysis and intelligent automation.

Python

Used to write powerful scripts that automatically collect data from the cloud APIs and insert it into the local database for processing and display.

PHP & MySQL (XAMPP)

Provides a solid backend to pull data from the MySQL database and serve it to the frontend, while also handling user authentication and other server-side tasks.

HTML / CSS / JS

Builds the modern user interface that displays data fetched by PHP and sends user commands via JavaScript to the control layer of the system.

Node-RED

Acts as the central messaging and logic hub. It receives commands from the web interface and translates them into instructions for the PLC hardware.

PLC S7-1200

The on-site control brain that reliably executes commands from Node-RED, controlling physical hardware like pumps, fans, and lights with industrial precision.
